# VibeStudying Public Site & Releases

이 저장소는 VibeStudying의 public 배포 저장소입니다.

역할은 두 가지입니다.

- GitHub Pages 홈페이지 제공
- Mac direct 배포용 릴리즈 자산 보관

## 홈페이지

- GitHub Pages: `https://dongwook9805.github.io/vibestudying-releases/`

현재 루트 `index.html`에는 다음 내용이 함께 들어 있습니다.

- 제품 소개 랜딩 페이지
- Mac direct 다운로드 안내
- 지원 정보
- 개인정보 처리방침

## Mac 다운로드

최신 Mac direct 배포본은 [Releases](https://github.com/dongwook9805/vibestudying-releases/releases) 페이지에서 받을 수 있습니다.

배포 자산:

- `.dmg`: 신규 설치용
- `-mac.zip` + `latest-mac.yml`: 앱 내부 자동 업데이트용

## 설치 방법

1. 최신 `.dmg` 파일을 다운로드합니다.
2. 다운로드한 파일을 엽니다.
3. `VibeStudying.app`을 `Applications` 폴더로 드래그합니다.

## 채널 정책

- `Mac`: 홈페이지 direct 다운로드 + 앱 내부 자동 업데이트
- `iPad / iPhone`: App Store 채널로 별도 운영
