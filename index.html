How It Works

실제 공부 흐름을 그대로 눌러보며 따라가 보세요.

아래 세 장은 모두 실제 Camto 화면입니다. 카드 만들기, 가리기 학습, AI 질문 장면이 그대로 전환됩니다.

Camto 실제 사용 흐름 카드 만들기 → 가리기 학습 → AI 팁
실제 앱 화면
자료 위에서 바로 카드 만들기 실제 이미지 가리기 카드 모달에서 라벨 박스를 잡고 저장 직전까지 다듬는 장면입니다.
학습하기에서 바로 복습 가려진 구조를 하나씩 열면서 위치와 이름을 같이 외우는 실제 학습 화면입니다.
같은 카드 맥락에서 바로 질문 학습 중 AI를 열고 해부 팁을 받는 실제 카드 학습 화면입니다.
Workflow

아래 단계 카드를 누르면 장면이 바뀝니다.

설명보다 먼저 실제 앱 흐름이 보이게, 자료를 읽고 복습하는 장면 그대로 넣었습니다.

아래 단계 카드를 클릭해 보세요
자료를 보다가 바로 카드 생성으로 들어갑니다.

실제 이미지 가리기 카드 모달에서 필요한 구조만 골라 가리고, 박스를 다듬은 뒤 그대로 저장합니다.

  • 자료를 보다가 바로 카드 생성으로 이동
  • 가릴 구조 박스를 직접 조정하고 저장
  • 저장한 카드는 곧바로 학습 화면으로 이어짐
